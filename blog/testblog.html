Account Number Check

The acc_no() function checks if an account number already exists. 

            
def acc_no(a):
    f = open("bank.dat", "rb")
    found = False
    try:
        while True:
            rec = pickle.load(f)
            if rec[0] == a:
                found = True
    except:
        if not found:
            return 1
    f.close()
  • Reading Data: Uses pickle.load() to read records from the file.
  • Unique Check:Returns 1 if the account number is unique.
